# slog
A very simple pure C log library

------------------------------

1. Supported C89
2. Supported Windows and Linux
3. Supported log stacktrace
4. thread-safe

5. Not asynchronous

------------------------------

一个非常简单的纯C日志库，做它的目的是因为工作中遇到过这样的一些需求：  

1. 只是一个很简单的小项目，不想有过多的依赖
2. 支持C89
3. 需要的时候可以打印调用堆栈
